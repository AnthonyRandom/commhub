# CommHub

A lightweight, cross-platform communication platform for real-time messaging, voice, and video.

[![License](https://img.shields.io/badge/License-Source%20Available-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-1.2.1-green.svg)](https://github.com/AnthonyRandom/commhub/releases)

## Features

- Real-time text messaging with channels and direct messages
- Voice and video calls using WebRTC
- Server communities with multiple channels
- Friend system and user presence
- Modern, responsive interface
- Cross-platform desktop application

## Download

Official releases: [GitHub Releases](https://github.com/AnthonyRandom/commhub/releases)

**Server:** NestJS, PostgreSQL, Prisma, Socket.io, JWT  
**Real-time:** WebSocket, WebRTC
